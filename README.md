# Asake_Discography
A SQL database modeling Asake’s discography, streaming performance, and global charting analytics stats till dec 31 2025.

This project is a relational SQL database designed to model and analyze the discography of Asake, focusing on:
Album and song structure
Streaming performance (Spotify & Apple Music)
Global chart rankings
Geographic and language-based market analysis

The database simulates a real-world music analytics system, similar to those used by streaming platforms and record labels.
Design a normalized relational database
Analyze music streaming performance
Evaluate global market penetration
Explore cultural and language-based trends
Build a portfolio-ready SQL project
